Overview-->

This project demonstrates connecting a Java app to a MySQL database using IntelliJ IDEA for basic CRUD operations.

Steps to Integrate MySQL-->

Download MySQL Connector/J: Get the .jar file from the MySQL website.
Add to Project: Place the .jar in a lib folder and add it to your project dependencies in IntelliJ.
Set Database Connection: Update your connection details in your Java code.
Create Table: Run the necessary SQL command to set up your table in the database.

Running the App-->

Launch your application in IntelliJ while ensuring your MySQL Server is running!
